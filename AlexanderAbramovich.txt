Идея приложения: 
Fitness Helper - помощник в тренировках.
Функции приложения:
- справочная информация по тренировкам групп мышц, питании, спортивные добаки и т.д.
- набор оптимальных программ для разных групп людей
- возможность компоновать собственную программу тренировок
- статистика результатов
- личный профиль
- поиск напарника для тренировок
- лента спортивных новостей


Целевая аудитория:
1 категория - люди начинающие заниматься спортом. Для данной категории есть набор базовых программ тренировок плюс дополнительная полезной информации (теория, питание, и т.д)

2 категория - люди активно занимающиеся спортом. Для данной категории предусмотрена возможность составление собственной программы тренировок, статистика, поиск напарника для тренировок.
